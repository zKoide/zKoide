### 🚀 Projetos em Destaque

#### 🏋️‍♂️ [TrainSpot](https://github.com/zKoide) (SaaS Multi-tenant para Academias & CrossFit)
* **Backend:** Fastify (Node.js) + TypeScript, PostgreSQL e Prisma ORM.
* **Fintech & Integrações:** Integração da Asaas API com split automatizado de tarifas e assinaturas recorrentes (PIX e Cartão de Crédito).
* **Mensageria:** Processamento assíncrono em background de tarefas pesadas usando filas com Redis e BullMQ.
* **Segurança:** Desenvolvido sob o conceito de *Security by Design*, usando autenticação JWT, validação rigorosa com Zod, logs estruturados com Pino e monitoramento com Sentry.

<br>

#### 🍔 [DashEats](https://github.com/zKoide) (SaaS para Gestão Gastronômica)
* **Real-time:** Comunicação de baixíssima latência via WebSockets (Socket.io) escalada horizontalmente sobre instâncias de Redis.
* **Backend:** API RESTful robusta em Node.js (Express) com banco de dados PostgreSQL (Neon).
* **Interface & UX:** SPA dinâmica em React 19 com Material UI, Styled Components, gráficos analíticos em Recharts e mapas interativos Leaflet.
* **Integração de IA:** Chatbot inteligente integrado à API do Google Gemini para triagem de atendimento e controle operacional de pedidos.
